## HTML

1. Насмотренность. Идем на сайт [https://html5up.net/](https://html5up.net/), изучаем все макеты, начиная с последнего до первого. Самые последние сделаны раньше, в них более читаемая структура
2. Из всего списка макетов выбрать 5 новых/последних, не слишком простых (галереи не подойдут, макет должен быть хотя бы на 2 экрана). Необходимо написать список элементов, которые на нем есть: можно на русском. Главное - список без вложенностей, плоский. Все слова с маленькой буквы, если их несколько, то нужно разделять дефисом. Пример:
    1. *шапка*
    2. *главный-баннер*
    3. *слайдер-с-продуктами*
    4. *форма-обратной-связи*
    5. *подвал*
    
    Если не понятно, что за блок или есть затруднения как назвать: **обращайте внимание на заголовок блока.**
    
3. Эти же 5 макетов сделать в виде древовидного списка т.е. взять уже готовый список из предыдущего и “раскрыть его”.
Пример:
    1. *шапка*
        1. *логотип*
        2. *навигация*
            1. *ссылка*
            2. *ссылка*
            3. *ссылка*
        3. *поле-поиска*
            1. *поле-ввода*
            2. *кнопка-поиска*
        4. *call-to-action кнопка*
    
    1. *главный-баннер*
        1. *текстовая-часть*
            1. *заголовок*
            2. *описание*
            3. *кнопка*
        2. *картинка*
    
    … и т.д.
    
4. Если писали на русском, то перевести на английский и добавьте точку перед каждым названием. Зайдите на сайт [https://codebeautify.org/pug-to-html-converter](https://codebeautify.org/pug-to-html-converter) и введите в поле слева то, что у вас получилось. Поздравляю, вы изучили 90% того, что нужно для создания сайта. Справа вам отобразится ваш список в виде html кода
5. Ознакомьтесь с основными HTML **тегами** ([https://html5book.ru/html-html5/](https://html5book.ru/html-html5/)) Вас интересуют:
    1. Р[азделы документа](https://html5book.ru/razdely-dokumenta/)
    2. [Списки](https://html5book.ru/html-lists/)
    3. [Ссылки](https://html5book.ru/hyperlinks-in-html/)
    4. [Формы](https://html5book.ru/formy/)
6. Перепишите ваш код, сгенерированный из прошлого задания, используя только что изученные теги. **Обратите внимание на правила создания списков, форм, таблиц, заголовков (особенно h1).** Если тег - это контейнер для других тегов, то поищите его название в списке тегов (например header, footer). Если такого тега в списке нет - значит это div. Также перенесите весь текст (в прошлых заданиях этого делать не нужно было)

Пример
    
    ```css
    .header {
    	width: 100%;
    	padding: 10px;
    	margin-bottom: 10px;
    	display: flex;
    }
    ```
    
2. Из курсов HTML Academy вы уже должны знать, как подключить файл со стилями к html файлу. Сделайте это. Изучите информацию о flexbox, grid и свойства width/height/padding/margin, ознакомьтесь с заданием размеров в пикселях, процентах, em, rem

    ```css
    /* Это контейнер для карточек, не сами карточки */
    .tarif-plan-wrapper {
    	display: flex;
    	padding: 12px 0;
    	gap: 10px;
    }
    ```
    
4. Зайдите в DevTools во вкладку Elements на странице с одним из макетов. Через инстумент поиска элементов наведите на один из блоков на странице и посмотрите его значения ширины, высоты, отступов и т.д

### Создание библиотеки работы со временем на JavaScript

---

#### **Цель проекта:**
Вы хотите создать библиотеку для работы со временем на JavaScript, которая будет включать функционал таймеров и секундомеров. Эта библиотека должна быть гибкой, удобной в использовании и поддерживать различные сценарии применения. В процессе разработки вы изучите ключевые концепции JavaScript, начиная с базовых (функции, объекты, классы) и заканчивая продвинутыми (асинхронное программирование, работа с DOM, интеграция с API).

---

#### **Что вы получите в итоге:**
1. **Библиотеку для работы со временем**, которая включает:
   - Таймеры с настраиваемой длительностью, повторениями и обратным отсчетом.
   - Секундомеры с возможностью фиксации промежуточных результатов (лапы).
   - Управление таймерами через консоль, HTML-интерфейс и клавиатуру.
   - Поддержку событий, Promise-based API и `async/await`.
2. **Глубокое понимание JavaScript**, включая:
   - Работу с асинхронным кодом (таймеры, Promise, async/await).
   - Манипуляции с DOM для создания интерактивного интерфейса.
   - Работу с API (загрузка данных, синхронизация времени).
   - Тестирование и TypeScript-типизацию.
3. **Практический опыт разработки**, который включает:
   - Постепенное усложнение задач.
   - Рефакторинг кода для улучшения читаемости и поддерживаемости.
   - Создание модульной и расширяемой архитектуры.

### **Этап 1: Базовые функции (Консоль)**

---

#### **1. Создать объект Timer**
**Описание:**  
Создать класс `Timer` с методами `start()` и `stop()`, которые выводят в консоль сообщения о запуске и остановке таймера.

**Пример входных данных:**
```javascript
const timer = new Timer();
timer.start();
timer.stop();
```

**Пример выходных данных:**
```
Timer started
Timer stopped
```

---

#### **2. Добавить параметр времени**
**Описание:**  
Добавить параметр `duration` в конструктор `Timer`, чтобы таймер автоматически останавливался через указанное время.

**Пример входных данных:**
```javascript
const timer = new Timer(5); // 5 секунд
timer.start();
```

**Пример выходных данных:**
```
Timer started
Timer stopped (через 5 секунд)
```

---

#### **3. Реализовать обратный отсчет**
**Описание:**  
Добавить вывод оставшегося времени в консоль каждую секунду.

**Пример входных данных:**
```javascript
const timer = new Timer(5);
timer.start();
```

**Пример выходных данных:**
```
Timer started
4 seconds left
3 seconds left
2 seconds left
1 second left
Timer stopped
```

---

#### **4. Добавить callback при завершении**
**Описание:**  
Добавить возможность передавать callback-функцию, которая выполнится по завершении таймера.

**Пример входных данных:**
```javascript
const timer = new Timer(5);
timer.start(() => console.log("Timer completed!"));
```

**Пример выходных данных:**
```
Timer started
Timer completed!
```

---

#### **5. Создать Stopwatch**
**Описание:**  
Создать класс `Stopwatch` с методами `start()`, `stop()`, `reset()` для измерения времени.

**Пример входных данных:**
```javascript
const stopwatch = new Stopwatch();
stopwatch.start();
setTimeout(() => stopwatch.stop(), 3000); // Остановить через 3 секунды
```

**Пример выходных данных:**
```
Stopwatch started
Stopwatch stopped: 3.00 seconds
```

---

#### **6. Реализовать форматирование времени**
**Описание:**  
Форматировать время в формате `мм:сс` (например, 02:30).

**Пример входных данных:**
```javascript
const stopwatch = new Stopwatch();
stopwatch.start();
setTimeout(() => stopwatch.stop(), 150000); // 2 минуты 30 секунд
```

**Пример выходных данных:**
```
Stopwatch started
Stopwatch stopped: 02:30
```

---

#### **7. Добавить историю замеров**
**Описание:**  
Сохранять все промежуточные результаты секундомера в массив.

**Пример входных данных:**
```javascript
const stopwatch = new Stopwatch();
stopwatch.start();
setTimeout(() => stopwatch.lap(), 2000); // Зафиксировать промежуточный результат
setTimeout(() => stopwatch.stop(), 5000); // Остановить через 5 секунд
```

**Пример выходных данных:**
```
Stopwatch started
Lap: 2.00 seconds
Stopwatch stopped: 5.00 seconds
History: [2.00, 5.00]
```


#### **11. Реализовать Promise-based API**

```javascript
timer.start().then(() => console.log("Timer completed!"));
```

**Пример выходных данных:**
```

Timer completed!
```

#### **12. Добавить async/await**
**Описание:**

**Пример входных данных:**
```javascript

  console.log("Timer completed!");
}
runTimer();
```

**Пример выходных данных:**
```
Timer started
Timer completed!
```



#### **14. Реализовать обратный отсчет с разными интервалами**


#### **15. Создать HTML-виджет таймера**
**Описание:**
